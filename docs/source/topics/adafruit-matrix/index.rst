Using the Adafruit LED Matrix Backpack
======================================

.. container:: pmslide

   Use the Adafruit LED Matrix Backpack to display innformation

   Update the Robot Configuration to add the Adafruit Matrix:

   * Upload "AdafruitLedMatrix.java" and "LedMatrix.java" from the src/
     directory

   * Rebuild and restart robot

   * | Under "I2C Bus 2"
     |    Add Type "Adafruit LED Matrix", name "matrix"
