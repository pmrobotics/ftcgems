Using a REV LED as driver feedback
==================================

.. container:: pmslide

   Use the REV LED to indicate when apriltag is seen.

   Update the Robot Configuration to add REV LED device:

   * | Under "Digital Devices"
     |    Port 0:  type "LED", name "ledr"
     |    Port 1:  type "LED", name "ledg"
